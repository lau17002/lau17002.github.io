Expense Tracking

FREE

This is a free feature included with the creation of an account. It allows clients to track expenses and to put them in a category. Included is a comprehensive list of totals in each category to allow for easy comparison between budgeted and actual expenses.

Budget Software

$49/year ($4/month)

The software by itself can be purchased on a yearly or monthly basis. This includes access to software through the client portal and templates to start creating a personal budget. Templates are customizable and there is an option to create them from scratch.

Consulting services

$10 per meeting

The first meeting with a client will be an hour long to set up expectations and the initial budget. Each meeting after that will be for thirty minutes to review success and revise as needed. Can be held virtually or physically depending on the clients needs.

Package

$39/month

This package includes all services tied to the budget software with weekly consulting services (1st meeting an hour long then 30 minutes per meeting). A $8 fee will be charged for any meetings canceled within a 48-hour period before the meeting starts.
